
# Intelligent Credit Card Fraud Detection System

## Major Project Overview
This project implements an intelligent credit card fraud detection system using machine learning techniques to identify fraudulent transactions from highly imbalanced financial data.

## Technologies
Python, Scikit-learn, XGBoost, SMOTE

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Place creditcard.csv in data/ folder
3. Run: python src/train_model.py
